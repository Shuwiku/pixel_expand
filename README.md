# Pixel Expand

## Использование

```python
from pixel_expand import expand

# Увеличит изображение в 4 раза
expand(
    image_in="image.png"
    image_out="image_exp.png",
    scale=4
)
```

Смотрите также [примеры использования](https://github.com/Shuwiku/Shuwiku/tree/main/pixel_expand).
